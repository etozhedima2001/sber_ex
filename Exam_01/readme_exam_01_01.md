## Exam_01_01
При написании программы была допущена ошибка. Необходимо исправить синтаксические, стилевые и логические ошибки, не изменяя работу программы. В конце ответа не должно быть переноса на новую строку. 
```C
#include <stdio.h>
int main(void) {
   int x = 0;
   scanf("%d", x);
   printf("%05d", x);
   return 0;
}
```
